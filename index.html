English description:

This program helps you for dynamically create web pages, or applications graphical interfaces via Javascript. Focused in Html objects (tag). This program able to merging the nodes for an 'user defined' objects in a document window. This Member (objects) are the gateway of the node properties, which you modify immediately with the members. You can add new atributes, events, and delete of a node simply and quick. Change the page imagine or pattern with an simple click-event or other comand. Use basic elements easily or load complete javascript tempelates, from the page's server if you need.

Experience for this app: (?beginner) HTML-advanced Javascript-basic

Hun. description:

Ez egy Javascript program, ami dinamikussá teszi a Weboldalak és alkalmazások grafikus felületeinek elkészítését. Fókuszban a Html elem (tag), amely objektumok mellé a program lehetővé tesz Javascript Member (objektumok), hozzárendelését, melyek a dokumentum ablak-on lesznek regisztrálva, és teljes kontrollt ad a felhasználójuknak, felettük. A membereken keresztül a legtöbb html csomópont tulajdonságai egyszerűen módosíthatóak, illetve eseménykezelő, és új tulajdonságok renelhetők az adott elemhez. Képes egyszerre több csomópont 'függőségi' (child-parent) viszonyának, az oldal mintázatának (pattern) dinamikus átalaítására. Ezenkívül a script képes emulálni az objektumokat az oldalról, vagy akár független javascriptben írt témákat, forrásokat betölteni, és inicializálni akár egy egyszerú click, vagy más parancsra.

A használathoz szükséges ismeretek: (?kezdők vegyék figyelembe) HTML-fejlett tudás (pl.: nem figyelmeztet ha ahibásan adtuk meg egy elem attribútumait); Javascript-alapvető ismerete szükséges.

When the meridian has setting up in the page the following tool create html elements, you want.

all of has properties by this 'member' object, and it's allow to modify them, query them immediately.

The other case for set a new element, has given as an easiest form, now we can set the value var member=Member({}),
and call the setMember() function, with the arguments, first is the name of element an the second is the attribute set.

Use selector:

This application has an selector handler, which called Use, this enable to get one, or more element refernece
from a document, with an unique attribute name of the specified. If the second argument is true, then returns
self (object),with the element, otherwise returns the selected element only;

Use('attributename').element;

Use('attributename');

Emulate:

The emulate process is an advanced selector, because it creates member objects from the captured html nodes.
It's same as to the created elements. The usage is not to much differ than the Use selector.

Emulate('attributename1',...);

LoadModul manager:

Loads external css or javascript files, from a folder, that not before linked in a document.
The css files only needed the path name of file, but the javascript files are not passed automaticaly.
The javascript file extension must be .jsx file.
The external function calls for different ways, from the file. 1. call in the sourcefile,
2. call as a string with an parameter array,
3. and call in a general function named as string.

script load without arguments:
the handler defined and call in the external file.

loadModul('./path/filename.jsx');

or

loadModul('./path/filename.jsx','hadlername');

or script loaded with handler, and has a parameterlist

loadMudul('./path/filename.jsx','hndlername',[parameters,...]);
