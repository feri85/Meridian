English description:

This program helps you for dynamically create web pages, or applications graphical interfaces via Javascript. Focused in Html objects (tag). This program able to merging the nodes for an 'user defined' objects in a document window. This Member (objects) are the gateway of the node properties, which you modify immediately with the members. You can add new atributes, events, and delete of a node simply and quick. Change the page imagine or pattern with an simple click-event or other comand. Use basic elements easily or load complete javascript tempelates, from the page's server if you need.

Experience for this app: (?beginner) HTML-advanced Javascript-basic

Hun. description:

Ez egy Javascript program, ami dinamikussá teszi a Weboldalak és alkalmazások grafikus felületeinek elkészítését. Fókuszban a Html elem (tag), amely objektumok mellé a program lehetővé tesz Javascript Member (objektumok), hozzárendelését, melyek a dokumentum ablak-on lesznek regisztrálva, és teljes kontrollt ad a felhasználójuknak, felettük. A membereken keresztül a legtöbb html csomópont tulajdonságai egyszerűen módosíthatóak, illetve eseménykezelő, és új tulajdonságok renelhetők az adott elemhez. Képes egyszerre több csomópont 'függőségi' (child-parent) viszonyának, az oldal mintázatának (pattern) dinamikus átalaítására. Ezenkívül a script képes emulálni az objektumokat az oldalról, vagy akár független javascriptben írt témákat, forrásokat betölteni, és inicializálni akár egy egyszerú click, vagy más parancsra.

A használathoz szükséges ismeretek: (?kezdők vegyék figyelembe) HTML-fejlett tudás (pl.: nem figyelmeztet ha ahibásan adtuk meg egy elem attribútumait); Javascript-alapvető ismerete szükséges.
